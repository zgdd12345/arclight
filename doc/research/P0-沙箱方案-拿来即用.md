# P0 本地沙箱方案（拿来即用版）

> 按用户指令「尽量使用现有工作、不自研，等产品成熟后再自研」。**零自研隔离原语**：不写 SBPL、不写 Landlock/seccomp helper、不 vendoring bwrap。我们只写薄 `SandboxService` 编排层。
> 依据：`research/data/topic-sandbox-security.json`（已调研的现成沙箱清单与许可证）。Codex(gpt-5.5) 读真实代码后产出。

## 决策一览

| 场景 | 方案（现成） | 许可证/形态 | 我们写什么 |
|---|---|---|---|
| **本地执行（默认）** | **nono**（Landlock/Seatbelt 封装，JSON profile，credential proxy，原子快照，Merkle 审计） | Apache-2.0，`brew install nono` | 只生成 profile + 编排调用 |
| **远程 opt-in** | **Vercel Sandbox**（首选，Firecracker，Next.js 原生）/ **E2B**（备选，持久 session/模板） | SaaS SDK | ~100-200 行 adapter |
| **浏览器侧轻量代码** | **Pyodide**（WASM） | 开源 | 直接 `loadPyodide()` |
| **兜底** | **Docker**（`--network none --read-only`） | 现成 | 一条 docker run |
| **未来候选（暂不用）** | Microsoft MXC（MIT，统一 JSON policy+TS SDK） | early preview，官方声明**暂不作安全边界** | 仅跟踪 |

## 本地：nono profile（`.arclight/sandbox/profiles/p0-local.json`）

```json
{
  "name": "arclight-p0-local-code", "version": 1, "backend": "auto",
  "filesystem": {
    "default": "deny",
    "read": ["/bin","/sbin","/usr","/lib","/lib64","/etc","${workspace}"],
    "write": ["${workspace}","${workspace}/.arclight/tmp","${workspace}/.arclight/cache"],
    "deny": ["${home}/.ssh","${home}/.aws","${home}/.config","${home}/.gnupg",
             "${home}/Library/Keychains","${workspace}/.git","${workspace}/.arclight/secrets"],
    "protect_names": [".ssh",".aws",".config",".env",".env.local",".npmrc",".pypirc"]
  },
  "network": { "default": "deny", "allow": [] },
  "process": { "inherit_policy_to_children": true, "kill_process_tree_on_exit": true },
  "resources": { "wallclock_ms": 120000, "cpu_seconds": 60, "max_pids": 128,
                 "max_open_files": 256, "max_stdout_bytes": 524288 },
  "credentials": { "mode": "proxy", "inject": [], "deny_raw_env": true },
  "snapshot": { "enabled": true, "root": "${workspace}", "mode": "atomic" },
  "audit": { "enabled": true, "format": "merkle-jsonl",
             "path": "${workspace}/.arclight/audit/sandbox-${run_id}.jsonl" }
}
```
网络批准后只走代理白名单（`network.proxy.url` + `allow_hosts`），不直接全开。

## 本地：从 Bun 调用（零自研隔离）

```ts
const child = Bun.spawn([
  "nono","run","--profile", profilePath,
  "--set", `workspace=${workspace}`, "--set", `home=${process.env.HOME ?? ""}`,
  "--set", `run_id=${runId}`, "--audit-log", auditPath, "--snapshot", "atomic",
  "--", "bash", "-lc", command
], { cwd: workspace, env: sanitizedEnv, stdout: "pipe", stderr: "pipe", stdin: "pipe" });
```
**凭证硬边界直接用 nono credential proxy**：沙箱内零真实 secret（`inject:[]`、`deny_raw_env:true`），需要凭证的动作经 proxy socket 由内核按动作签名放行。审计用 nono 的 Merkle JSONL，内核只存引用（runId/auditPath/merkleRoot/profileHash）。

## 远程 opt-in adapter（现成 SDK，仅 ~100-200 行）

```ts
// 网页优先首选 Vercel Sandbox
import { Sandbox } from "@vercel/sandbox";
const sandbox = await Sandbox.create({ timeout: "5m", resources: { vcpus: 2, memory: "4gb" } });
await sandbox.files.write("/workspace/package.json", pkg);
const proc = await sandbox.runCommand({ cmd: "bash", args: ["-lc","npm test"], cwd: "/workspace" });
for await (const c of proc.stdout) emitStdout(c);
// 备选 E2B：需持久 session/模板时用
```

## 唯一自研 = 薄编排层（接缝，非轮子）

```ts
type SandboxKind = "local-nono" | "remote-vercel" | "remote-e2b" | "browser-pyodide" | "docker-fallback";
interface SandboxService {
  probe(): Promise<{ ok: boolean; backend: SandboxKind; reason?: string }>;
  run(req: SandboxRunRequest): Promise<SandboxRunResult>;   // 流式回灌 ArcEvent
  cancel(runId: string): Promise<void>;
}
```
ArcEvent 回灌：`tool.progress(sandbox.start/stdout/stderr)` → `tool.output(spillRef)` → `session.error(SANDBOX_UNAVAILABLE)`。

## 失败/降级阶梯
1. `local-nono`（默认）→ 2. `docker-fallback`（`docker run --rm --network none --read-only --pids-limit 128 -v $WS:/workspace:rw ...`）→ 3. `remote-vercel/e2b`（用户显式 opt-in）→ 4. **拒绝执行**（nono+Docker 都不可用 / 需网络未批 / 需真实凭证注入 / 要挂 docker.sock·SSH agent·Keychain）。**绝不偷偷裸跑。**

## 验证 spike（2-3 人日）
- **D1** `brew install nono` + `nono doctor --json`（确认 backend 非 silent-degraded）；验 workspace 可写 / `~/.ssh`·`~/.aws` 不可读 / `.git` 不可改 / 默认网络失败 / audit 生成。
- **D2** 真实 workload：`npm test` / `pytest` / `npm run build` 能跑且不被改写 HOME 大面积误伤；取消能杀进程树。
- **D3** Bun 集成稳定捕获 stdout/exit + `AbortSignal` 取消 + nono 不可用自动探 Docker + 都不可用返回 `SANDBOX_UNAVAILABLE`。
- **Go/No-go**：mac+linux 至少一个平台 nono 强约束通过、另一平台有 Docker 兜底；secrets 不可读；网络默认关；无遗留进程。

## 成熟度风险（诚实）
nono 处"早期稳定/设计合作伙伴"阶段：CLI/profile schema 可能变；Linux Landlock 需内核 ≥5.13（旧 LTS 静默降级，**probe 必须拦截 silent degradation**）；macOS Seatbelt 网络粒度粗（细控靠 proxy）；credential proxy / Merkle audit / atomic snapshot 这些高级能力**必须实测，不按宣传写进承诺**。回退：Docker + opt-in 远程；若 nono 稳定性不足，未来再评估 MXC（成熟后）或极薄自研 wrapper——**不是现在**。

## 对主蓝图的影响
